🎮 Easy Host Server

Hébergez votre serveur de jeu Steam en un clic, sans prise de tête.

Voir la source officielle Liste complète des jeux compatibles

Comment ça marche ?

  1. Construisez l’image Docker
  2. Lancez le conteneur avec l’AppID du jeu
  3. Profitez de votre serveur !
docker build -t easy-steam-server .
docker run -it easy-steam-server 740

Projet open-source. Fait avec ❤️ pour la communauté gaming.
